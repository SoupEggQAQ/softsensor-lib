# Softsensor Library(softsensor-lib)
softsensor-lib 是一个面向深度学习研究者的开源库，专注于软测量（Soft Sensor）建模与实验分析。
该项目致力于为软测量任务提供统一、易扩展、便于复现的实验框架，支持研究人员快速开展模型训练、评估与对比实验。
---
## 项目简介
在工业过程建模与软测量研究中，研究者通常需要反复进行数据处理、模型搭建、训练评估和实验管理。
softsensor-lib 面向这一需求进行设计，旨在提供一个结构清晰、扩展方便的实验平台，帮助用户高效开展以下任务：

软测量单步预测

软测量多步预测

软测量多目标值预测

多种深度学习模型的统一训练与评估

自定义模型的快速集成与实验复现

本项目特别适合以下用户：

软测量与过程工业方向研究者

## 快速开始

### 1. 准备数据

将软测量数据置于 ```./dataset``` 目录

### 2. 安装

1. 克隆本仓库

2. 创建新的 Conda 环境

3. 安装核心依赖

### 3. 训练与评估

1. 运行```./scripts```脚本文件

### 4. 开发自定义模型

1. 将模型文件放入 ```./models```, 可参考 ```./models/GRU```

2. 在```./models/__init__.py```中导入模型

3. 在```./exp/exp_basic.py```的```Exp_Basic.model_dict```中注册新模型。

4. 在```./scripts```下创建对应的运行脚本
